GFXBCTS
=======

A simple bytcode VM written in typescipt.

## How to run

Build everything using

`npm build`

Open 

[`dist/index.html`](dist/index.html)

in a browser.

## Tests

`npm test`

## Test coverage

`npm run coverage`

Then open [coverage/index.html](coverage/index.html)
