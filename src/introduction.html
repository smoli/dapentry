How to use dapentry

dapentry allows you to create rich, data driven visualisations without writing code. This example ...

On this page, we'll introduce all tools available that will enable you to create similar visualisation.

Drawing shapes

First, let's create some basic primitives: Circles, rectangles and lines. On the top of the designer you'll find a toolbar. Click any of the buttons and draw the according shape.

All drawing tools in dapentry are "click-based". That means you click for each point of the shape you're drawing instead of holding down the mouse key and dragging a shape. We think this will lead to a more relaxed way of working.

Using hotkeys

All actions of the designer can be accessed via hotkeys. Learning these hotkeys will allow you to work quicker and concentrate more on your design and less on the tool.

The hotkeys for the three primitives above are

  • C - Circle
  • R - Rectangle
  • L - Line

All tools behave the same, no matter if you activated them using a hotkey or the toolbar. In this designer we've hidden the toolbar. Try drawing the with those primitives again.
