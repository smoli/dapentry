An Introduction to dapentry

dapentry allows you to create rich, data driven visualisations without writing code. On the following pages, we'll introduce the tools available that will enable you to create visualisations yourself.

Drawing shapes

First, let's create some basic primitives: Circles, rectangles and lines. On the top of the designer you'll find a toolbar. Click any of the buttons and draw the according shape.

All drawing tools in dapentry are "click-based". That means you click for each point of the shape you're drawing instead of holding down the mouse key and dragging a shape.

Using hotkeys

All actions of the designer can be accessed via hotkeys. Learning these hotkeys will allow you to work quicker and concentrate more on your design and less on the tool.

The hotkeys for the three primitives above are

  • C - Circle
  • R - Rectangle
  • L - Line

All tools behave the same, no matter if you activated them using a hotkey or the toolbar. In this designer we've hidden the toolbar. Try drawing with those primitives again.

Transformations

You can of course transform objects like you may be used to from other tools. The basic transformations for moving (translation), rotating and scaling are supported.

There are of course hotkeys for these as well

  • G - Move (grab)
  • T - Rotate (turn)
  • S - Scale

You can try these on the rectangle in the designer if you like. These expect you to grab one of the handles and drag it around.

The pivot point for scaling and selecting is selected automatically to be the opposite point.

Snapping

One important feature is snapping. You can snap to points of other objects when creating and transforming objects.

The canvas you're drawing on has snap points as well. This allows you to create drawings that always fit into the canvas and change when the canvas size changes.

Try drawing again. You'll see snap points appear for the canvas, circle and every object that you have drawn. You can also manipulate the objects.

Since the toolbar is gone, here's a reminder of the hot keys:

  • C - Circle
  • R - Rectangle
  • L - Line
  • G - Move
  • T - Rotate
  • S - Scale
